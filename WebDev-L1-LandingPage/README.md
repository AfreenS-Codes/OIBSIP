# Modern Landing Page

A clean, responsive, and visually polished landing page built using HTML5 and CSS3. This project demonstrates the fundamentals of modern web design through a structured layout, consistent typography, and responsive styling without relying on JavaScript.

## Overview

The landing page is designed with a modern aesthetic, featuring a sticky navigation bar, an engaging hero section, well-organized content areas, and a responsive layout that adapts seamlessly across desktop, tablet, and mobile devices.

## Features

- Responsive layout using Flexbox
- Sticky navigation bar
- Hero section with call-to-action
- Multiple structured content sections
- Modern typography and consistent spacing
- Cohesive colour palette
- Mobile-friendly design
- Clean and semantic HTML structure

## Tech Stack

- HTML5
- CSS3
