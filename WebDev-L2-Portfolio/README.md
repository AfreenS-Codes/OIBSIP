# Personal Portfolio Website

A modern and responsive personal portfolio website designed to showcase my skills, projects, achievements, and contact information. The website serves as a digital resume while reflecting a clean and professional design philosophy.

## Overview

This portfolio highlights my journey as a Software Engineering student with an interest in web development, problem solving, UI design, and building meaningful technology.

The interface follows a minimal and elegant design with responsive layouts, structured content, and smooth user experience.

## Features

- Responsive navigation
- Hero introduction section
- About Me
- Skills showcase
- Featured Projects
- Contact section
- Responsive design
- Clean and modern interface
