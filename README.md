## About This Repository

This repository documents my work for the **OIBSIP Web Development internship track**.
Every task is treated as a real deliverable — not a throwaway exercise — with attention
given to visual design, responsive behavior, accessibility, semantic markup, and code
that's actually readable six months from now.

Each project lives in its own self-contained folder, follows a strict naming convention,
and ships with its own README covering the brief, the approach, and how to run it locally.

## Design Philosophy

Across every task in this repository, a few principles stay constant:

- **Clean architecture over clever shortcuts** — code that's easy to extend, not just easy to demo
- **Design and engineering as one discipline** — visual polish isn't an afterthought
- **Simplicity over complexity** — the simplest solution that fully meets the brief wins
- **Responsive by default** — every project is verified on desktop, tablet, and mobile viewports
- **No dependencies unless they earn their place** — plain HTML/CSS/JS unless a task calls for more

## Running Any Project Locally

Every project in this repository is a static site — no build step, no package manager required.

```bash
# 1. Clone the repository
git clone https://github.com/<your-username>/OIBSIP.git

# 2. Move into the project you want to run
cd OIBSIP/WebDev-L3-TemperatureConverter

# 3. Open index.html directly in your browser
#    or serve it locally for the best experience:
python3 -m http.server 5500
# then visit http://localhost:5500
```
