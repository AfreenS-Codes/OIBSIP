# Temperature Converter

A modern temperature conversion web application built using HTML5, CSS3, and Vanilla JavaScript. The application converts values between Celsius, Fahrenheit, and Kelvin while providing real-time validation and handling edge cases such as absolute zero.

## Overview

The project focuses on combining clean user interface design with accurate temperature conversion logic. It features a minimal monochrome theme, responsive layout, and user-friendly validation messages to ensure a smooth experience across all devices.

## Features

- Convert between Celsius, Fahrenheit, and Kelvin
- Input validation
- Absolute zero validation
- User-friendly error messages
- Responsive design
- Minimal monochrome interface
- Rounded values for improved readability
- Clean and accessible layout

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Validation

The application validates:

- Empty input
- Non-numeric values
- Absolute zero violations

Supported temperature limits:

| Unit | Minimum Value |
|------|---------------|
| Celsius | -273.15°C |
| Fahrenheit | -459.67°F |
| Kelvin | 0K |
